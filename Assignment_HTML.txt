1. Are the HTML tags and elements the same thing?
ANS:- No, HTML tags and elements are not the same thing. HTML tags are used to hold the HTML element whereas HTML element are used to holds the content. HTML tag starts with < and ends with > and whatever written within <> are elements. For example: <h1></h1>. Here h1 is the HTML tag and when it is kept in between <> i.e. <h1></h1> then it is called HTML element.

2. What are tags and attributes in HTML?
ANS:- HTML tags are used to hold the HTML element. All HTML elements can have attributes in them. Attributes are used to provide additional information about element. It is always specified in the start tag. Example: <a href :“”>< /a>. Here <a></a> is the HTML element and href: “” which is inside the element is the HTML attribute.

3. What are void elements in HTML?
ANS:- Void elements are the self-closing elements in HTML. These elements do not need a closing tag. These elements may have attributes in them. For eg: <br>, <img>, <hr>, etc. these are some of the void elements.

4. What are HTML Entities?
ANS:- HTML entity is used to display some symbols or characters that would otherwise be interpreted as HTML codes. It is a piece of text that begins with an ampersand (&) and ends with a semicolon (;). Eg: For using less than ‘<’ symbol entity name: ‘&lt;’ needs to be used.

5. What are different types of lists in HTML?
ANS:- There are 2 different types of lists in HTML, they are: 
i.	Ordered List: It starts with <ol> and ends with </ol>. In ordered list items are listed in numbers.
ii.	Unordered List: This list starts with <ul> tag. Here items are listed in bullets.
iii.	Nested List: A nested list is a list inside another list.
iv.	Definition List: A definition list or description list is a list of terms, with a description of each term. The <dl> tag defines the description list, the <dt> tag defines the term, and the <dd> tag describes each term.

6. What is the ‘class’ attribute in HTML?
ANS:- The class attribute specifies one or more class names for an element. Class is represented by ‘.’ Symbol.  It can be used on any HTML element. 

7. What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements?
ANS:- The only difference between ‘id’ attribute and the ‘class’ attribute is that “id” is unique in a page and can only apply to at most one element, while “class” selector can apply to multiple elements. Also the ‘id’ attribute is represented by ‘#’ symbol and the ‘class’ attribute is represented by ‘.’ Symbol.

8. What are the various formatting tags in HTML?
ANS:- To adjust the formatting of text, HTML offers a variety of predefined elements. The text can be highlighted, made superscript or subscript, or given other text styles, such as bold, italic, or emphasised, using formatting. Some of the formatting tags are: <b> tag for to make the text bold, <i> tag to make the text italic, <sup> tag to superscript a text whereas <sub> to subscript a text, <del> tag is used to delete the text, <mark> tag is used to highlighting a text, etc.

9. How is Cell Padding different from Cell Spacing?
ANS:- Cell padding basically defines the space present between a table cell’s border and the content present in it. Cell spacing basically defines the space present between individual adjacent cells.

10. How can we club two or more rows or columns into a single row or column in an HTML table? 
ANS:- We club two or more rows or columns into a single row or column by using Colspan and Rowspan attributes. We can merge two or more table cells in a column using the colspan, whereas to merge two or more row cells, use the rowspan attribute. This attributes are used inside the <td> tag.

11. What is the difference between a block-level element and an inline element? 
ANS:- A block-level element always starts on a new line, this element always takes up the full width available where an inline element does not start on a new line. This element only takes up as much width as necessary.

12. How to create a Hyperlink in HTML? 
ANS:- To create a hyperlink in an HTML, we use the <a> and </a> tags, which are the tags used to define the links. Just add the URL for the link in the <a href=” ”> and the hyperlink will be created.

13. What is the use of an iframe tag? 
ANS:- An HTML iframe is used to display a web page within a web page. The <iframe> tag specifies an inline frame. An inline frame is used to insert another document within the current HTML document. The ‘src‘ attribute is used to specify the URL of the document that occupies the iframe.

14. What is the use of a span tag? Explain with example? 
ANS:- The <span> tag is an inline container used to mark up a part of a text, or a part of a document. The span tag is used for the grouping of inline elements & this tag does not make any visual change by itself. Span is very similar to the div tag, but div is a block-level tag and span is an inline tag.

15. How to insert a picture into a background image of a web page? 
ANS:- To add a background image in HTML element, use the HTML style attribute and CSS image property in the HTML tags. For eg: <p style= “background-image: url(xyz.jpg)”></p>

16. How are active links different from normal links?
ANS:- Normal links are links which are there on the page and have not been clicked yet. Active links are those links, which have just been clicked at that instant.
